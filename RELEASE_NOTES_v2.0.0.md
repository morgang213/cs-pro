### üöÄ Quick Start

#### Linux/macOS:
```bash
# Download and extract
wget https://github.com/morgang213/cs-pro/releases/download/v2.0.0/cybersec-terminal-v2.0.0.tar.gz
tar -xzf cybersec-terminal-v2.0.0.tar.gz
cd cybersec-terminal-v2.0.0/

# One-command install
./install.sh

# Launch web terminal
python terminal_web.py
```

#### Windows:
1. Download `cybersec-terminal-v2.0.0.zip` from releases
2. Extract the ZIP file
3. Double-click `install.bat`
4. Run `python terminal_web.py`

#### Python Package Installation:
```bash
pip install cybersec-terminal==2.0.0

# Run commands
cybersec           # Launch interface selector
cybersec-web       # Start web terminal
cybersec-terminal  # Start CLI terminal
```
